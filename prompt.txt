@Entity
@Table(name = "ability_definition")
@Getter
@Setter
@NoArgsConstructor
@AllArgsConstructor
public class AbilityDefinition implements Models {

  @Id
  @Enumerated(EnumType.STRING)
  private AbilityName code;

  @Column(nullable = false)
  private String name;

  @Column(nullable = false)
  private String descriptionKey;

  @Enumerated(EnumType.STRING)
  private AbilityType type;

  @ManyToMany(fetch = FetchType.EAGER, cascade = {CascadeType.MERGE, CascadeType.REFRESH})
  @JoinTable(
      name = "ability_definition_notes",
      joinColumns = @JoinColumn(name = "ability_definition_code", referencedColumnName = "code"),
      inverseJoinColumns = @JoinColumn(name = "note_id", referencedColumnName = "id")
  )
  private List<Note> notes;
  
  @ElementCollection
  @Enumerated(EnumType.STRING)
  private List<AbilityName> requirements;

  @ElementCollection
  @Enumerated(EnumType.STRING)
  private List<AbilityName> unlockables;

  @Enumerated(EnumType.STRING)
  private RequirementType requirementType;
}

Mistura Base:
Inchiostro del crepuscolo
Ingredienti: Alcohol, Nigredo
Effetti: L’Alchimista può creare un inchiostro
invisibile e il relativo solvente rivelatore,
seguendo una ricetta pratica con ingredienti
reali oltre i cartellini Ingredienti.
L’inchiostro consente di scrivere messaggi
che non sono visibili a occhio nudo.
Per motivi di gioco, è obbligatorio marcare
i quattro angoli fronte e retro del foglio
con un simbolo evidente (per esempio
con asterischi), a segnalare la presenza del
messaggio invisibile.
Ricetta per l’inchiostro invisibile:
1 cucchiaino di bicarbonato di sodio
2 cucchiai d’acqua
(Mescolare fino a ottenere un liquido con cui
scrivere.)
Come usare il Solvente?
Prima di usare il solvente su tutto il foglio è
obbligatorio bagnare con il solvente uno
dei quattro angoli del foglio.
Se compare un qualsiasi simbolo nascosto
allora il foglio conterrà messaggi nascosti,
contrariamente non servirà usare il
solvente sul foglio.
Ricetta per il solvente rivelatore:
1/2 cucchiaino di curcuma in polvere
2 cucchiai di alcool o igenizzante liquido
(Mescolare e applicare sul foglio per far
emergere il messaggio.)
